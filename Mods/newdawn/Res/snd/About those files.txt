This folder contain .wav files, to be used with SN receiver.
All new files starts with letter 'z', to not mess with internal files. Please stick to that rule.
Sounds that DOESNT start with 'z' are for New Sound Engine.

Due to only 12 characters filepath limit, names are roughly shortened, so there is translation to Format SN:
	newd0...newd99 ~ newday     +
	newm0...newm99 ~ newmonth   +
	neww0...neww99 ~ newweek    +
	chest0...chest9 ~ chest     +
	arti0...arti99 ~ treasure   +
	lvlu0...lvlu99 ~ nwherolv   +
	fmine0...fmine9 ~ flagmine  +
	ques0...ques99 ~ Quest      +
	draqn0...draqn9 ~ Dragon    +
	view0...view99 ~ view       +
	piqu0...piqu99 ~ pickup01..07 +
	myst0...myst99 ~ Mystery    +
	milit0...milit9 ~ Military  +
	roqe0...roqe99 ~ Rogue      +
	gzbo0...gzbo99 ~ Gazebo     +
	gprot0...gprot9 ~ GetProtection.wav +
	exp0...exp999 ~ Expernce    +
	genie0...genie9 ~ Genie     +
        nomd0...nomd99 ~ Nomad      +
